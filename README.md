
### Instalação
Abra terminal


```bash
cd txmIe8
```

Use esse comando na pasta raiz:

```bash
bower install txmIe8 --save
```

ou

```bash
npm install angucomplete-ie8 --save
```


### Start

Para start da aplicação, entre no diretótio 'www e inicialize server com este comando:

```bash
cd www
python -m SimpleHTTPServer
```
